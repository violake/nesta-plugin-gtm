Installation
------------

To use this plugin just add it to your Nesta project's `Gemfile` and
then install it with Bundler:

```
    $ echo 'gem "nesta-plugin-gtm"' >> Gemfile
    $ bundle
    $ cp [nesta_path]/views ./
    $ cp [nesta-plugin-gtm_path]/views/*.* ./views/

```
